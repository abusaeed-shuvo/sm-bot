# sm-bot
Scalar Modular discord bot for easy modification and deployment

A scalable Discord bot template using slash commands.

## Features
- Modular architecture
- Slash commands
- User info command

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy env:
   ```bash
   cp .env.example .env
   ```

3. Add your bot token

4. Run:
   ```bash
   python main.py
   ```

## Commands

/user -> Get user info
/avatar -> Get user's avatar
